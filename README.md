## What is The Conqueror ?
The Conqueror is an Battle Royale board game using multiple differents assets : cards (attack, defense, traps), physical players and mosnters cards.
Uses logic and plan ahead to gather epic gears, defeat monsters and the other players to win the game.

This project is mainly built using HTML, CSS, and JavaScript as part of a school project.

## Installation & Start
- Download the folder or use
``` php
git clone https://github.com/LeoVigin/the-conqueror.git
```
- Launch it using Go Live (or any local development server).
